# UAE Property Agent

Real Estate Property Management System.

## Features
- Budget and location input
- UAE-style bright property UX/UI
- Property recommendations
- Serious buyer filtering
- Visit scheduling / booking system
- Smart chat interface
- Follow-up simulation for email and WhatsApp/SMS

## Run Locally
```bash
npm install
npm run dev
```

Open the local URL shown in terminal, usually:
`http://localhost:3000`

## Build
```bash
npm run build
```
